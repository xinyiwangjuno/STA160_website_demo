📌 About the Project

This project was created as part of STA160 to demonstrate how statistical analysis and visualization can reveal insights from complex datasets. We explore descriptive statistics, hypothesis testing, and modeling techniques to uncover meaningful patterns.

🧪 Methods

Our analysis includes data cleaning, exploratory data analysis (EDA), and regression modeling. Below is an overview of the workflow:

  1. Data preprocessing and feature selection
  2. Exploratory analysis and visualization
  3. Model building and validation
  4. Result interpretation
